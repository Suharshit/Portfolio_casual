# Project Context: Suharshit Singh Portfolio

## Core Overview
A high-performance, premium developer portfolio designed to showcase 
full-stack engineering skills, performance optimization, and 
advanced interaction design.

## Technical Foundation
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript (Strict Mode)
- **Execution**: pnpm (Preferred Package Manager)
- **Deployment**: Vercel (Analytics Integrated)

## Aesthetic & UI
- **Styling**: Tailwind CSS 4 + Vanilla CSS Variables
- **Theme**: Premium Dark (Abyss, Foam, Wave, Snow palette)
- **Scroll**: Lenis Smooth Scrolling
- **Interactions**: Custom Interactive Cursor, Magnetic Elements

## Component Architecture
- **Hero**: Parallax typography and scroll-driven ghost name.
- **Projects**: Scroll-triggered stacking card "peel" effects.
- **Education**: GSAP-powered horizontal timeline visualization.
- **Skills**: Interactive scatter chips with category filtering.
- **Certificates**: 3D tilt-card grid with badge verification.
- **Achievements**: Animated count-up statistics.
- **Contact**: Marquee dividers and giant parallax text.

## Current State
- **Performance**: Optimized with Next.js 15 features and lazy loading.
- **Animations**: Heavy use of Framer Motion variants and GSAP ScrollTrigger.
- **Assets**: CDN-ready assets with responsive image handling.
- **Dependencies**: Iconify/Lucide for iconography, GSAP for timelines.

## Recent Focus
- Linting and build stability across Next.js 15 environment.
- Refinement of Neumorphic design elements and glassmorphism.
- Integration of advanced scroll-driven animations and cursor physics.
